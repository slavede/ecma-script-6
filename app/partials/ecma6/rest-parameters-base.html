In case you have optional number of parameters you need to send into function

  • Using '...' before name of last argument of function
  • Replaces arguments variable, but with few differences:
    • arguments parameter contains ALL parameters passed to function, '...'-parameters are only those which are not named parameter (the rest of the parameters)
    • arguments parameter is not real Array

var sum = function(start, ...additional) {
	additional.forEach(function(val) {
		start += val;
	});
	return start;
}

AppLogger.info(sum(1,5,7,9,10));
AppLogger.info(sum(3));
