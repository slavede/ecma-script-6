Key/value object where key can be anything (any other object).

  • When constructing map, value must be iterable and each value must constist of key-value pair (for example, array of 2-element arrays where first element is key)
  • Key equality is based on '===' operator. And NaN is matched to NaN even though NaN !== NaN
    • var object1 = {
	name : 'Slaven'
},
object2 = {
	surname : 'Tomac'
},
object3 = {
	surname : 'Tomac'
},
myMap = new Map([
	[object1,'ObjectWithNameOnly'],
	['myKey1',object1],
	[object2,object1]
]);
  • .get(key) method returns value for the key, or undefined if it doesn't exist
    • AppLogger.info(myMap.get(object1));
AppLogger.info(myMap.get('myKey1'));
AppLogger.info(myMap.get(object3));
AppLogger.info(myMap.get(myMap.get('myKey1')));
  • .delete(key) method deletes object with given key and returns true/false based if key existed or not
    • AppLogger.info(myMap.get(object1));
myMap.delete(object1);
AppLogger.info(myMap.get(object1));
  • .entries() returns iterator for iterating map. Each .next() call returns object with key-value pair (see Generators for more details how iterators work).
    • var iterator = myMap.entries();
AppLogger.info(iterator.next());
AppLogger.info(iterator.next());
AppLogger.info(iterator.next());
  • .forEach() returns iterator which contains an array of key-value pair for each element
    • myMap.forEach(function(val) {
	AppLogger.info(val);
});
  • .has(key) returns true/false if object has (or hasn't key). .set(key, value) sets value under specified key
    • AppLogger.info(myMap.has(object3));
myMap.set(object3, 'setObject');
AppLogger.info(myMap.has(object3));
  • .keys() returns iterator over the keys.
    • var keysIterator = myMap.keys();
AppLogger.info(keysIterator.next());
AppLogger.info(keysIterator.next());
AppLogger.info(keysIterator.next());
  • .values() returns iterator over the values
    • var valuesIterator = myMap.values();
AppLogger.info(valuesIterator.next());
AppLogger.info(valuesIterator.next());
AppLogger.info(valuesIterator.next());
  • Map object has default iterator which can be used in for..of iterator. It iterates over key-value pairs (see Generators-for..of for more details about default iterators)
    • for (var [key, val] of myMap) {
	AppLogger.info('key');
	AppLogger.info(key);
	AppLogger.info('val');
	AppLogger.info(val);
}
