Why this page?

I created this page to show new feature of EcmaScript6 in action. Since still (now it's May of 2015) EcmaScript6 is not supported in all browsers I had to use Babel in some cases (even Babel doesn't support everything).

EcmaScript 6 in Action

Go to EcmaScript6 Examples to see examples. On the left side there will be links to each new feature in EcmaScript6 and sub-link which will be native (source code not converted with Babel) and Babel (source code converted with babel). On the right side of each page you will see logger dump of the code running. Source code you will see in the middle will be code how it should be written for EcmaScript6, but based on link (babel or not), it will run different code. In pages where babel code is being run you will be able to click "Show Babel" to see in what code it converted initial EcmaScript6 code. Some pages will have only native in case Babel conversion is the same.

BabelJS

Since ES6 is not fully supported in all of the browsers some libraries/tools were created which convert ES6 to ES5 code. One of those is BabelJS. I won't be giving introduction in BabelJS. For more details you should take a look here
