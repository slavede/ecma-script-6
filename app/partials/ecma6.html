EcmaScript6 Examples

What will you find here?

  • Template Strings give us new way of interpolation of strings
  • Let keyword is a new 'var' (blocked scoped var)
  • Arrow functions allow us functions without parenthesis and different way of binding 'this'
  • Classes are new way of inheritance in JavaScript
  • Modules (only in Firefox) will allow simple import/export of JavaScript module (replacement for AMD and CommonJS modules)
  • Default parameters allow us to assign some default values to parameters that are passed into functions
  • Rest parameters will allow us to use optional number of parameters when defining functions
  • Generators are new way for iterating through data
  • Promises give use built-in way for async programming
  • Maps will replace usage of Objects as map objects (anything can be key now)
  • Sets are classic Set object (they are now part of JavaScript)
  • Const (only in Chrome) keyword will allow us to define constants inside our program
  • Typed Array (only in Firefox) allows declaration of arrays with predefined type of elements inside an array
  • Proxy (only in Firefox) allows us to define custom behavior for built-in operators such as property lookup, function invocation etc

What's AppLogger

  • AppLogger is wrapper around console which is used to display console messages in browser.
  • In examples you can hover with your mouse of AppLogger.info entries to see where's that in code. Works vice-versa as well.
